    Parameters

    • a: Vector3d

      First vector.

    • b: Vector3d

      Second vector.

    A new vector that is perpendicular to both a and b, has Length == a.Length * b.Length * sin(theta) where theta is the angle between a and b. The resulting vector is oriented according to the right hand rule.
